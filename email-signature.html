How to install in Outlook Mac:
  1. Open this file in Safari (not Chrome).
  2. Click anywhere inside the dashed box, then press Cmd+A to select all.
  3. Press Cmd+C to copy.
  4. In Outlook → Preferences → Signatures, click + to create a new signature.
  5. Click inside the signature editor and press Cmd+V to paste.
  6. Give the signature a name and save.
Note: the logo file (oyl-logo-signature.png) must remain in the same folder as this file. Once your website is live, switch to a hosted image URL for wider compatibility.
